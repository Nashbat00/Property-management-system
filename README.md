# HomeLink - Property Management System

## Project Overview

HomeLink is a modern, web-based property management platform designed to streamline communication and administrative tasks between building managers and residents. The system enables managers to handle dues, payments, maintenance requests, and announcements while providing residents with a transparent and user-friendly interface to interact with their building management.

HomeLink is built using a client-server architecture with **React** as the frontend single-page application (SPA), **Supabase** as the backend-as-a-service (BaaS), and **Tailwind CSS** for styling. The application is deployed on **Vercel**.

- **Frontend:** React (SPA)

- **Deployment:** Vercel
- **Communication:** REST API + WebSocket (real-time subscriptions)
