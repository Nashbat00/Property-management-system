# HomeLink - Property Management System

## Problem Statement

Traditional building management relies on paper-based records, group chats, and informal communication channels. This leads to opaque financial records where residents cannot verify dues or payment history, scattered communication where announcements get lost, manual payment tracking prone to errors, unstructured maintenance requests that are forgotten or delayed, and no audit trail for resolving disputes.

HomeLink is a web-based property management platform that solves these problems by providing a centralized, real-time digital system for managing monthly dues, tracking payments, handling maintenance requests, and broadcasting announcements within a residential building. The system uses a client-server architecture with React 18 as the frontend SPA, Supabase as the backend-as-a-service (BaaS), Tailwind CSS for styling, and Vercel for deployment.
